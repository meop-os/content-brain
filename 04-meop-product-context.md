# Meop Product Context (for content creation)

## What Meop Is
Company OS that signs deals. AI infrastructure that replaces the ops team for companies doing outbound sales.

NEVER call it: "AI SDR", "AI assistant", "chatbot", "automation tool"

## What's Live

### Audit Report Generator (meop.live/audit/generate)
Internal sales tool. Enter any company domain. Runs live OSINT (DMARC/SPF/subdomains/breaches/GitHub). Fills branded HTML report in under 60 seconds.
Use case: Drives $1,500 audit sale.

### Benware Demo (meop.live/demo)
Primary sales tool / lead magnet. Passively collects device fingerprint, GPU, location, network IP via WebRTC, camera/mic count, battery. Zero permissions, zero clicks, under 2 seconds. Shows visitor what was captured. Pitches hardware-layer security fix.

### Dashboard (meop.live)
Primary dev dashboard. CRM, pipeline, content studio, intel, brain interface.

### Admin Mission Control (meop.live/dashboard/admin/mission-control)
Bloomberg-terminal style panel showing pipeline, goals, task queue, brain pulses, active sessions.
