## Platform Specs

| Spec | Value |
|------|-------|
| **Post length** | Up to 3,000 characters (first 140 visible before "see more") |
| **Image** | 1200 x 1200 (square) or 1200 x 627 (landscape) |
| **Carousel/PDF** | Up to 300 pages, 1080 x 1080 or 1080 x 1350 per slide |
| **Video** | Up to 10 min, horizontal or vertical |
| **Best time** | Tue-Thu, 7-8am or 12pm (test for audience) |
| **Frequency** | 5x/week minimum for growth |
| **Algorithm priority** | Comments > Shares > Reactions > Clicks |

## LinkedIn vs Other Platforms

| Dimension | TikTok/IG | LinkedIn |
|-----------|-----------|----------|
| **Tone** | Funny, chaotic, casual | Confident, insightful, real |
| **Goal** | Awareness, virality | Credibility, relationships, deals |
| **Audience** | General public, Gen Z/Millennial | Investors, executives, operators, founders |
| **Content** | Entertainment-first | Value-first (insight, lesson, framework) |
| **CTA** | Follow, share, comment | Connect, DM, engage in thread |
| **Success metric** | Views, followers | Meaningful connections, DMs, opportunities |

## LinkedIn Content Types

### 1. Insight Post (2x/week)
One clear insight about AI, startups, or the industry.

**Format:**
```
[Bold opening line — the insight]

[2-3 lines of context]

[The "so what" — why this matters]

[CTA — question to drive comments]
```

### 2. Lesson Post (1x/week)
Something we learned building MEOP. Real, specific, useful.

**Format:**
```
[What happened — one line]

[The context — 2-3 lines]

[What we learned — the actual lesson]

[How you can apply this — make it useful]

[CTA — "What's your experience with this?"]
```

### 3. Category Post (1x/week)
Establishing AVAI as a category. Thought leadership.

**Format:**
```
[Bold statement about the category/market]

[Why this matters now — timing, trends, data]

[Our perspective — how MEOP sees this]

[CTA — debate-worthy question]
```

### 4. Build-in-Public Post (1x/week)
Real updates on building MEOP. Transparent.

**Format:**
```
[Milestone or update — one line]

[What we did — specific, not vague]

[What's next — forward momentum]

[CTA — "What would you build next?"]
```

### 5. Carousel / Document Post (1x/week)
Deep-dive visual content. Highest save and share rate on LinkedIn.

**Slide structure:**
```
Slide 1: HOOK — Bold claim or question
Slide 2-7: CONTENT — One point per slide, clean design
Slide 8: SUMMARY — Key takeaways
Slide 9: CTA — Follow + comment prompt
Slide 10: ABOUT — One-liner about MEOP + handle
```

## Walker's Voice Rules (Mandatory for LinkedIn)

### Format
- One line per thought
- Skip a line between every line
- 5-10 words per line target
- No paragraphs. Ever.
- Bullets only for intentional lists

### Tone
- Conversational, not performative
- Confident, not arrogant
- Direct, not aggressive
- Smart but never showing off

### Word Choice
- Simple words. "Use" not "utilize." "Start" not "commence."
- If a 12-year-old wouldn't say it, cut it.
- No jargon unless it's THE jargon you're coining (AVAI)

### BANNED Phrases
- "Here's the thing"
- "Let me be honest"
- "Hot take"
- "Unpopular opinion"
- "Most people don't realize"
- "Leverage" as a verb
- Any overused LinkedIn-ism
- "Agree?"
- "Thoughts?"

### Humor
Dry, understated, self-aware. Seasoning, not the main course.

## LinkedIn CTA Best Practices

**Best CTAs for LinkedIn (by effectiveness):**
1. Open-ended question → drives comments
2. "Repost if you agree" → extends reach
3. DM prompt → "DM me [keyword]" for direct leads
4. Tag prompt → "Tag someone who needs this"
5. Link → only when genuinely valuable (algorithm penalizes external links)

**Pro tip:** Put links in the FIRST COMMENT, not the post body. Algorithm doesn't penalize comments with links.
